# Fundamentos de Java para Principiantes

## Resumen del Curso
Este curso introductorio de aproximadamente 1 hora está diseñado para enseñar a estudiantes de secundaria (12 a 17 años) los fundamentos de la programación con Java, incluyendo la programación orientada a objetos (POO). ¡No requiere descargas ni instalaciones! Todo el entorno de desarrollo y las pruebas se realizan directamente desde el navegador web.

* **Duración total**: ~60 minutos
* **Público Objetivo**: Estudiantes de 12 a 17 años sin experiencia previa en programación.
* **Prerrequisitos**: Ninguno.
* **Herramientas necesarias**: ¡Solo tu navegador web! (Chrome, Firefox, Safari, Edge).
* **Repositorio de Código Fuente**: [Enlace a tu GitHub de GuardiAnts](https://github.com/upc-pre-202610-1asi0729-11896-guardiant/java-fundamentals-course-guardiants)

## Secuencia de la Lección

### Lección 1: ¿Qué es Java y la Programación? (6 minutos)
* **Descripción**: Aprende qué es la programación y por qué Java es un excelente lenguaje para empezar a crear aplicaciones.
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/EB4pRFLtXMo)
* **Conclusiones clave**: Java se usa para aplicaciones y juegos; los programas son instrucciones para computadoras.
* **Empieza a programar**: [Abrir Replit Hola Mundo](https://onlinegdb.com/your-gdb-id)

### Lección 2: Variables y Tipos de Datos (8 minutos)
* **Descripción**: Aprende a almacenar datos en Java usando variables de forma sencilla.
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/WJXtJPe2G8I)
* **Conclusiones clave**: Uso de `int` y `String`. Declaración de variables como `int edad = 15;`.
* **Práctica**: [Clic para programar]((https://onlinegdb.com/your-gdb-id)

### Lección 3: Estructuras de Control: Decisiones y Bucles (10 minutos)
* **Descripción**: Toma decisiones en tu código con `if-else` y repite tareas usando bucles `for` y `while`.
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/rL1CZgFZQpg)
* **Conclusiones clave**: `if (edad >= 12) { ... }` y estructuras iterativas.
* **Práctica**: [Clic para programar en JDoodle]((https://onlinegdb.com/your-gdb-id)

### Lección 4: Métodos, Entrada/Salida e Introducción a la POO (10 minutos)
* **Descripción**: Crea código reutilizable con métodos, recibe datos del usuario usando la biblioteca `Scanner` e introduce los conceptos de clases.
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/o1p4oi1AJHw)
* **Conclusiones clave**: Uso de la biblioteca `Scanner` para interactuar con la consola y definición de un método simple.
* **Práctica**: [Haz clic para practicar en OnlineGDB]((https://onlinegdb.com/your-gdb-id)

### Lección 5: Fundamentos de POO y Programas Sencillos (10 minutos)
* **Descripción**: Construye clases con atributos y comportamientos, aplicando encapsulación. Desarrollaremos un registro de "Estudiantes".
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/T8pZhuDrxz4)
* **Conclusiones clave**: Instanciación de objetos, uso de constructores y ocultamiento de datos con getters.
* **Proyecto final**: [Crea tu aplicación POO]((https://onlinegdb.com/your-gdb-id)

### Lección 6: Estructuras de Datos Simples - Arrays (8 minutos)
* **Descripción**: Aprende a guardar múltiples datos del mismo tipo en una sola variable utilizando Arrays o arreglos, ideal para hacer inventarios.
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/iVkuYh4gM1s)
* **Conclusiones clave**: Creación de un `String[]`, uso de `.length` para conocer el tamaño, y recorrido de listas con bucles `for`.
* **Práctica**: [Clic para programar el inventario]((https://onlinegdb.com/your-gdb-id)

### Lección 7: Mini Proyecto - Juego de Adivinanzas POO 
* **Descripción**: Unimos todos los conocimientos (variables, bucles, Scanner y clases) para crear una aplicación interactiva donde el jugador debe adivinar un número secreto.
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/Rgbkf2rdZYw)
* **Conclusiones clave**: Integración de la biblioteca `Scanner` con bucles `while` dentro de un método encapsulado en una clase independiente.
* **Práctica**: [Juega y modifica el código]((https://onlinegdb.com/your-gdb-id)

### Lección 8: Mejores Prácticas y Siguientes Pasos
* **Descripción**: Consejos para evitar errores comunes de compilación, cómo leer la consola para probar el código y recursos para seguir aprendiendo.
* **Enlace**: [Ver la lección en YouTube](https://youtu.be/6Nrv-VSta6g)
* **Consejos clave**: Identar el código, no olvidar los puntos y comas, y explorar la documentación oficial de Java.
